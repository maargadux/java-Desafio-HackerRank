# 🧠 Weird or Not Weird (Java)

Desafio 2 - Hacker Rank (Java IF ELSE) - desafio com condicoes

## 🚀 Descrição

O programa lê um número inteiro e determina se ele é **"weird"** ou **"not weird"** com base em regras de intervalo.

## 📌 Regras implementadas

* Se número ≤ 5 → Not Weird
* Se número entre 6 e 20 → Weird
* Se número > 20 → Not Weird

## 🛠️ Tecnologias

* Java
* Scanner (entrada de dados)
* Estruturas condicionais (`if`, `else if`, `else`)
* Eclipse

## ▶️ Como executar

```bash
javac Weirdo.java
java Weirdo
```

## 💡 Exemplo

Entrada:

```
3
```

Saída:

```
Not Weird
```
